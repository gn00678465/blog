資料型別 Map

sass map

注意Sass Maps 是 Sass 3.3 版才有支援

Sass Maps 提供變數類似於 json 的資料組合，也是 key/value 的結構，可提升程式可讀、變數群組性。

寫Sass Maps的時候，要特別注意幾點：

  1. 外層要使用小括號。
  2. 最後一個key/value後面的逗號可以省略。
  3. key的名稱不能重複，否則會編譯錯誤。
  4. key/value的名稱可以是許多資料類型(數字、字串、布林等等)。

sass:map

  1. map-get($map,$key)
  2. map-merge($map1,$map2)
  3. map-remove($map,$key)
  4. map-keys($map)
  5. map-values($map)
  6. map-has-key($map,$key)

map-get($map,$key)

取出 $map 裡指定的 $key 所關連的 value。

map-merge($map1,$map2)

合併多個 $map。

map-remove($map,$key)

從 $map 裡面刪除一個移除指定的 $key。

map-keys($map)

取出所有的 $key。

map-values($map)

取出所有的 value。

map-has-key($map,$key)

瀏覽 $map 裡面是否有 $key 值，有則回傳true，沒有便回傳false。
