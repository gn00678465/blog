資料型別 List

Sass 中 List 就像一般 array，是用來存放沒有鍵名的變數，但當要從陣列裡取得資料的時候只能透過索引（1, 2, 3, 4）。

要注意的是在 Sass 中，List 從 1 開始算起，而不是正常程式界的 0。

list 可以使用的函數

  1. nth
  2. index
  3. join
  4. append
  5. zip
  6. length

nth

list.nth($list, $n)
透過 nth(變數, 索引) 你能夠取得 List 裡的資料。

index

list.index($list, $value)
index的其中一種做法可以拿來算你所選擇的變數排序。
