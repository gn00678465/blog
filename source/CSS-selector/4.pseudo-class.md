## 錨點(Anchor)
{% note info %}
主要讓元素可以跟使用者作互動，算是一種可以將元素即時套用樣式的效果。
{% endnote %}

    <tbody>
        <tr>
        <td>:link</td>

        <td>:visited</td>
        </tr>
        <tr>
        <td>:active</td>
        <td>:focus</td>
        <td>:target</td>
        </tr>
    </tbody>
</table>
{% endraw %}

## 結構
{% note info %}
主要有`-child`和`of-type`兩類
`child`:父元素內的子元素都是 child。
`of-type`:如果子元素交錯混雜，可用於選取固定元素類型
{% endnote %}
