基本選擇器

群組式宣告

  • 若希望不同的選擇器中可以套用相同的樣式，最容易的作法是使用以下語法。
  • 可以將任意選擇器群組在一起，沒有任何限制。
  • 透過群組，可以將某類型的樣式壓縮在一起，可以得到更簡潔的樣式表。
Syntax

通用選擇器 (*)

  • Universal selector以星號(asterisk)*表示，該選擇器可以與任何元素搭配
Syntax

元素選擇器 (elementName)

  • 元素選擇器（型態選擇器）依照 Node 節點名稱選取匹配的 Element 元素。
Syntax

Class 選擇器 (.classname)

  • Class 選擇器依照具有class屬性來選取匹配的 Element 元素。
  • Class 選擇器在一個 HTML 文件中可以被使用多次。
  • Class 的宣告法，是先放一個句點 (.)，之後再列出選擇器名稱。
Syntax

ID 選擇器 (#idname)

  • ID 選擇器依照具有ID屬性來選取匹配的 Element 元素。
  • ID是唯一的，當出現多組具有相同ID屬性的元素時，只會指向第一個匹配的元素。
  • ID 的宣告法，是先放一個井字號 (#)，之後再列出選擇器名稱。
Syntax

屬性選擇器 ([attr=value])

  • 屬性選擇器可以依據元素的屬性或屬性值來選取匹配的 Element 元素。
  • 需使用中括號([])將屬性或屬性值包覆。
Syntax
上述語法屬性以及屬性值需完全匹配。
依據部分屬性值選擇
選擇器 描述
[attr~=value] 選擇完整特定文字屬性值的元素。
[attr^=value] 選擇具有特定文字開頭屬性值的元素。
[attr$=value] 選擇具有特定文字結尾屬性值的元素。
[attr*=value] 選擇包含特定字串屬性值的元素。
特定屬性選擇
[attr|=value] 選擇等於特定文字或者是其開頭屬性值的元素。
